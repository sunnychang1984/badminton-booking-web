# Badminton Booking Web

## Project Overview

The Badminton Booking Web application allows users to book badminton courts online. It simplifies the booking process for users and provides easy management for booking administrators.

## Features
- User Registration and Login
- Court Availability Check
- Online Booking System
- Booking History
- Admin Management Console

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MongoDB

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/sunnychang1984/badminton-booking-web.git
   ```
2. Navigate to the project directory:
   ```bash
   cd badminton-booking-web
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
- Visit the application at `http://localhost:3000`
- Users can register an account or log in to book courts.
